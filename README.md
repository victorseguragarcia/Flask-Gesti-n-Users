# Flask-Gestion-Users

## Para ejecutar se necesita estar en el directorio raiz del programa:

## Ejecutar pip install -r .\requirements.txt (si es posible en venv)

## Ejecutar python run.py y debería aparecer en http://127.0.0.1:5000
